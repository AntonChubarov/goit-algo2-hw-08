# goit-algo2-hw-08

## GOIT Design and Analysis of Algorithms Homework 8

### Requirements

- Python 3.10+
- Git

### Usage

Clone this repository:
```shell
git clone https://github.com/AntonChubarov/goit-algo2-hw-08.git
```

Navigate to project directory:
```shell
cd goit-algo2-hw-08
```

Run the tests:
```shell
python3 -m unittest
```
